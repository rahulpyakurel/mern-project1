Comparision Operator

        == -> equal to
        === -> equal value and equal data type
        != -> not equal
        !== -> not equal value or not equal data type
        > -> greater than
        < -> less than
        >= -> greater than or equal to
        <= -> less than or equal to
        ? -> ternary operator

        
Logical Operator

        && -> logical and
        || -> logical or
        !  -> logical not
